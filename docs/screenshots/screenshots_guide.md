# Portfolio Screenshots Guide

To showcase BusinessPilot AI on your portfolio or GitHub repository, we recommend capturing screenshots of these pages:

---

## Recommended Screenshots

1. **`01_dashboard.png`**
   - **Page**: `📊 Executive Dashboard`
   - **View**: Tab 2 (`📈 Lead Funnel Analytics`)
   - **Capture**: The metrics cards and active Plotly charts (Lead Prioritisation Funnel, Revenue by Industry, Conversion Probability).

2. **`02_timeline.png`**
   - **Page**: `📊 Executive Dashboard`
   - **View**: Tab 1 (`🚀 Workspace & Execution`)
   - **Capture**: The live **Agent Execution Timeline** trace console showing the step durations and tool runs.

3. **`03_explainability.png`**
   - **Page**: `📊 Executive Dashboard`
   - **View**: Tab 2 (`📈 Lead Funnel Analytics` -> bottom section)
   - **Capture**: The expanded **Lead Score Explainability Deep-Dive** scorecard showing point-by-point contributions for a selected company.

4. **`04_kanban.png`**
   - **Page**: `📊 Executive Dashboard`
   - **View**: Tab 4 (`📋 Follow-up Task Board`)
   - **Capture**: The interactive Kanban Board displaying task cards in `To Do`, `In Progress`, and `Completed` columns.

5. **`05_telemetry.png`**
   - **Page**: `🔧 MCP Observability`
   - **Capture**: The list of registered MCP Servers, exposed tools list, and the live tool invocation latency telemetry table.

6. **`06_history.png`**
   - **Page**: `📜 Pipeline History`
   - **Capture**: The list of historical runs showing run IDs, times, leads, revenue, and report availability.

7. **`07_architecture.png`**
   - **Page**: `🏛️ System Architecture`
   - **Capture**: The system architecture diagram page displaying components description.
